\doxysubsection{Detailed Description}
Smart Parking Sensor. 

This class is used for implementing the Smart Parking Sensor

The parking sensor consists of\+:
\begin{DoxyItemize}
\item one ultrasonic sensor (used for measuring distance to objects)
\item one piezo buzzer (used to alert us when we are too close to an object) 
\end{DoxyItemize}
